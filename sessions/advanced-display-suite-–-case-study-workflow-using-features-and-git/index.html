As a proof of concept and step up from the precursor session, The power of Display Suite (D7 & D6), I’d like to presented a recent site that was launched in D6 using all of the goodies that Display Suite has to offer. I was principle site programmer and had some DS help from Mr. Chellman.

This session should be considered intermediate to advanced and will most definitely be showing code. Consider it an organized teardown of the site in a “How it was done” approach to see how you can use Display Suite to spec content layout, use features to multiply your awesome skills over node types, and have your Drupal site quickly sharing the custom layout and settings on the content that you chose.

Git will be a sideshow compared to the Features/DS main event in the session, but we will be using git to apply our changes and check them into Version Control; a must for making sure that all of your changes are gonna show up in the right places.

No animals will be harmed during the session and nun chucks are optional but not required.
