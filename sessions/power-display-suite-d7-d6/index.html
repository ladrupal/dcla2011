The power of Display Suite (D7 & D6)

Join stevenator, chellman, and BTMash for some good old node theming fun as we delve into the pros and cons of the wonderful Display Suite package of modules. This one hour session will be a precursor to the session “Advanced Display Suite – case study of workflow using Features and Git”.

We will focus on Display Suite 7 out of the box and the great things that can be accomplished in theming and laying out content with some simple browser configuration and css. At the end of the session (time permitting), Chellman will discuss using the new D7 feature of custom templates that allow more adventurous Drupaleros to break out of the typical Header, 3 Column, Footer template that is offered through DS.

The beginning of the session will be a brief introduction for anyone who is not familiar with the power of Display Suite. We will gauge the level of the group when starting and model the talk around questions and the materials that we have prepared.

If you are a site builder and have not seen this module suite then this talk is for you…break out of the tpl craze for a little and see what can be offered…come drink the DS kool-aid :)!
