I don’t work on sites without these modules. This is just basic clean up of the admin area for developers/maintainers, to get rid of common headaches. Make permissions administration easily manageable (including multi-select checkboxes). Make the module administration page much more manageable (no more scrolling through tons of them or using ctrl F to locate the module you want to activate/deactivate). And of course: Teleport. I don’t even use admin menu anymore, teleport allows you to click ctrl M on any page, when you are logged in, and an ajax enabled auto-complete search box shows up for you to begin typing the section you want to go to, and boom, all the relevant sections start showing up for you to teleport to.
I added Masquerade and backup & Migrate, both also essential admin tools.

This session was a bit of a hit last year and I think it helped a lot of beginners over some common hurdels.
It should be quite quick again, this is what it will be like (For the Drupal 6 part of the demonstration):

Step 1: Download & upload all needed modules (listed above) to module folder.

Step 2: Enable Teleport.
This is how we will be navigating the admin area for the rest of the
admin cleanup.

Step 3: Enable Module filter.
This is a must have for simple module administration.

Step 4: Enable Permissions Select.
This will enable “select/deselect all” check boxes on the permissions
administration page.

Step 5: Enable Filter Permissions.
This turn the permissions administration section into something useful:)

Step 6: Enable Admin Role.
This module automatically assigns all permissions to an admin role
that can be added to any user. (We need to disable and re-enable a
module to see it take effect).

Step 7: Enable Backup & Migrate.
This allows for quick, easy, backups.

Step 8 (if there’s time):
Create a new roll.
Create a user with that role.
Enable Masquerade.
Configure if for quick switching between that user and the admin.
Add the Masquerade Block to the side nav.
